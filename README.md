# 🏥 札幌医大 合格対策アプリ

和士くんの札幌医大合格を応援する問題集アプリです。

## 📱 スマホで使えるようにする手順（GitHub Pages）

### ステップ1：GitHubアカウントを作る
1. https://github.com にアクセス
2. 「Sign up」からアカウント登録（無料）

### ステップ2：リポジトリを作る
1. GitHubにログイン後、右上の「＋」→「New repository」
2. Repository name: `sapporo-med-quiz`
3. Public を選択
4. 「Create repository」をクリック

### ステップ3：ファイルをアップロード
1. 「uploading an existing file」をクリック
2. `sapporo-med-quiz` フォルダの中身を全て選択してドラッグ＆ドロップ
3. 「Commit changes」をクリック

### ステップ4：GitHub Pagesを有効にする
1. リポジトリの「Settings」タブ
2. 左メニューの「Pages」
3. Branch: main → / (root) を選択
4. 「Save」をクリック

### ステップ5：URLを和士くんに共有
数分後に以下のURLでアクセスできます：
```
https://あなたのユーザー名.github.io/sapporo-med-quiz/
```

## 📚 収録問題数
- 数学：8問（微積・確率・数列・複素数・ベクトル）
- 英語：6問（文法・語彙・読解・英作文）
- 理科：8問（生物・化学：免疫・遺伝・有機・平衡）
- 国語：5問（現代文・古文・小論文）

## ✨ アプリの機能
- 選択式問題：自動採点
- 記述式問題：模範解答＋採点ポイント表示
- 正答率の自動計算・記録
- 科目別結果表示
- スマホ最適化デザイン
